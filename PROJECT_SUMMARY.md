## 🚀 快速开始

### 本地运行

```bash
# 1. 安装依赖
bundle install

# 2. 启动服务器
bundle exec jekyll serve

# 3. 访问
open http://localhost:4000
```

### 创建文章

```bash
# 在 _posts 目录创建文件
# 格式: YYYY-MM-DD-title.md

---
layout: post
title: "文章标题"
date: 2025-01-10 10:00:00 +0800
categories: [分类]
tags: [标签]
excerpt: "摘要"
---

文章内容...
```

### 部署到 GitHub Pages

```bash
# 1. 初始化 Git
git init
git add .
git commit -m "Initial commit"

# 2. 推送到 GitHub
git remote add origin https://github.com/yourusername/yourusername.github.io.git
git push -u origin main

# 3. GitHub Actions 会自动部署
```
